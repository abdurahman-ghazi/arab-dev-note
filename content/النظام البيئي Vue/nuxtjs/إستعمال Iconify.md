# بِسْمِ اللَّـهِ الرَّحْمَـٰنِ الرَّحِيمِ

# طريقة استعمال iconify

اولا التثبيت 

```
yarn add nuxt-icon
```

يتم إضافتها إلى modules في داخل nuxt.config.ts

```
modules: [
	'nuxt-icon',
]
```

و نظيفها بكل سهولة 

```
<Icon name="mdi:youtube" size="96" color="#15CA82"  />
```
